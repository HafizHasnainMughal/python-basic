# 📖 About

This repository is designed for beginners who want to learn Python from scratch.

Every topic is explained in a simple and beginner-friendly way with easy-to-understand examples.

The purpose of this repository is to build a strong Python foundation before moving to advanced topics like Functions, OOP, File Handling, Data Structures, Data Science, Artificial Intelligence, and Machine Learning.

## 9. Conditional Statements

Decision making in Python.

Topics Covered

- if
- if-else
- if-elif-else
- Nested if
- Logical conditions

Example

